## 使用方法
1.下载 **`Plugin.php`** 文件

2.在你的博客服务器 **`usr/plugins/`** 目录中创建名为 **`MaintenanceMode`** 的文件夹

3.将 **`Plugin.php`** 文件上传至 **`MaintenanceMode`** 的文件夹中

4.登录你的博客后台，点击 **`控制台`** ➔ **`插件`**，即可启用该插件
